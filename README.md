# Theatre of Pain | Garry's Mod Gamemode

[![Watch the Demo](https://img.youtube.com/vi/AAxCqAm0SDo/0.jpg)](https://www.youtube.com/watch?v=AAxCqAm0SDo)

This repository contains custom gameplay logic scripted for a Garry's Mod gamemode commissioned in 2017.

### Project Overview
The project focused on implementing a tailored multiplayer experience based on a survival and roleplaying premise between factions. 

### Technical Implementation
* **Language:** Developed using **Lua**.
* **Role:** Responsible for translating the client's design requirements into functional scripts within the Garry's Mod environment.
* **Scope:** Included game rules implementation and engine-specific logic to ensure a stable gameplay experience.
